Git docs
========

Uma simples revisão do conteúdo da aula.


É interessante que durante as aulas os alunos repitam os comandos
para a melhor retenção.


Build
-----

O material pode ser gerado em pdf com o comando

```
make pdf

```

Para gerar um docx 

```
make doc

```
